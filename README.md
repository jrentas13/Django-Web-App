# Personal Django Web Portfolio

A website that showcases all the projects I have been working on.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Python
2. Install Django
3. Type "py manage.py runserver" in terminal
4. CTRL + left-click on link

Instructions for using the software:

1. Type "py manage.py runserver" in terminal
2. CTRL + left-click on link
3. Use nav bar or scroll down to view each section
4. Click on view project to be redirected to github repo
5. Fill out the Contact form to send me an email
